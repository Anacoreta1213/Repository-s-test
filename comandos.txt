Hijueputa

git log : para ver nuestra para ver las fotos(commit hechos), para ver cada versión;

git commit: seguir la versión de nuestro programa, toma una foto que puede ser regresiva si así lo queremos;

git diff: Muestra las los cambios efectuados en el archivo modificado;

.gitignore:Añadimos los archivos que queremos ignorar;

git app: nos permite además, añadir los cambios a la rama si estamos convencidos de guardar los cambios efectuados en los diferentes archivos;

git commit -m "file" : para evitar abrir la cansola del editor.

git branch: Nos permite ver, si existen claro, las diferentes ramos de nuestro proyecto, también conocidas como versiones;

git branch "nameBranch": Nos permite crear una nueva versión de nuestro proyecto;

git checkout "nameBranch": Da lugar a movernos entre las diferentes versiones; 

git remote add origin "Url de almacenamiento": Nos permite ubicar y subir nuestro proyecto a github desde la consola de git bash;

git push -u origin main: Es el comando utilizado para subir nuestro proyecto a github definitivamente; 

git clone "URL de orgien": Nos ubicamos en una carpeta y luego le damos enteren la consola git bash;
